# She Can Foundation Full Stack Website

This project was created as part of the Full Stack Development Internship task for She Can Foundation.

It is a full stack web application built using React, Node.js, Express, and MongoDB Atlas. The website is designed for an NGO and includes a responsive frontend along with a backend for handling user enquiries.

The project includes a contact form where users can submit their details and messages. Submitted enquiries are stored in MongoDB and can be viewed from the admin page.

## Features

- Responsive frontend design
- React based user interface
- Backend API using Express.js
- MongoDB Atlas database integration
- Contact/Enquiry form
- Admin page to view submitted enquiries
- Mobile responsive layout
- Clean NGO themed UI

## Installation

Clone the repository:

```bash
git clone <repository-link>
```

Frontend setup:

```bash
cd client
npm install
npm run dev
```

Backend setup:

```bash
cd server
npm install
npm start
```

## Environment Variables

Create a `.env` file inside the server folder and add:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
